# Icon Files

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any image editor. The icons should:

1. Be PNG format
2. Have transparent backgrounds
3. Use a recognizable symbol (like a magnifying glass, code brackets, or analysis chart)
4. Be simple and clear at small sizes

## Temporary Placeholder

Until you create custom icons, you can use any PNG files with the correct names and dimensions, or create simple placeholder icons using online tools like:

- Canva
- GIMP
- Photoshop
- Online icon generators

The extension will work without custom icons, but Chrome will show a default puzzle piece icon.
